# DCR Calculator（线圈直流电阻计算器）

## 项目简介
一款专为电源/磁性元件开发工程师设计的桌面工具，用于快速推算线圈规格与理论DCR（直流电阻）。

## 痛点解决
在器件选型或反向设计时，工程师常需根据现有料号/线圈参数推算目标规格，并进行DCR估算。

- 手动计算繁琐易错
- Excel表格维护不便，缺乏直观性

## 核心优势
- **🧠降低认知负担**：将经验公式集成于后台，输入关键参数即可快速得到结果，让工程师专注于设计本身。
- **🖥️清晰的图形界面**：告别晦涩的表格公式，所有操作一目了然。
- **⚠️智能错误提示**：实时校验输入数据的合法性，避免因参数错误导致的设计偏差。
- **📋预设模板**：内置标准输入模板，无需重复设置常用参数，开箱即用。
